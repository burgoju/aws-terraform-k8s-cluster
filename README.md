## Project Overview
Production-ready Kubernetes cluster on AWS with auto-scaling.

## Features
- Terraform Infrastructure as Code
- EKS Cluster with t3.small nodes
- Horizontal Pod Autoscaler
- Ingress Controller
- Zero-downtime deployments
- S3 + DynamoDB state management

## Technologies
Terraform, AWS, Kubernetes, Docker
